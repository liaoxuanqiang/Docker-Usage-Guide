## Docker 基本命令

```bash
docker #列出 Docker CLI 中可用的命令
docker <COMMAND> --help #列出特定命令的信息
docker image ls --all #列出计算机上的 docker 映像
docker container ls --all #列出计算机上的容器
docker ps -a #如果没有 -a 显示全部标志，则仅显示正在运行的容器
docker info #列出有关 Docker 安装的系统范围的信息
```
